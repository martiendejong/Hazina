# Hazina.ChatShared

## Purpose
Shared UI components and models for chat experiences.

## Getting Started
- Restore/build: `dotnet build src/Core/UI/Hazina.ChatShared/Hazina.ChatShared.csproj`
- Add as project reference if consumed from another project: `dotnet add <your>.csproj reference src/Core/UI/Hazina.ChatShared/Hazina.ChatShared.csproj`

## Usage
- Add reference: `dotnet add <your>.csproj reference src/Core/UI/Hazina.ChatShared/Hazina.ChatShared.csproj`
- Build: `dotnet build src/Core/UI/Hazina.ChatShared/Hazina.ChatShared.csproj`

## API Reference
- XML docs generated on build: `bin/Debug/net8.0/Hazina.ChatShared.xml`.
- Use IDE tooling or `dotnet doc`/Sandcastle/DocFX to render API docs if desired.
