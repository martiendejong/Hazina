# Hazina.LLMs.Client

## Purpose
Client wrapper for LLM interactions.

## Getting Started
- Restore/build: `dotnet build src/Core/LLMs/Hazina.LLMs.Client/Hazina.LLMs.Client.csproj`
- Add as project reference if consumed from another project: `dotnet add <your>.csproj reference src/Core/LLMs/Hazina.LLMs.Client/Hazina.LLMs.Client.csproj`

## Usage
- Add reference: `dotnet add <your>.csproj reference src/Core/LLMs/Hazina.LLMs.Client/Hazina.LLMs.Client.csproj`
- Build: `dotnet build src/Core/LLMs/Hazina.LLMs.Client/Hazina.LLMs.Client.csproj`

## API Reference
- XML docs generated on build: `bin/Debug/net8.0/Hazina.LLMs.Client.xml`.
- Use IDE tooling or `dotnet doc`/Sandcastle/DocFX to render API docs if desired.
