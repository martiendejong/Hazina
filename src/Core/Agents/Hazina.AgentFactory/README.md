# Hazina.AgentFactory

## Purpose
Factory and dependency wiring for generator agents.

## Getting Started
- Restore/build: `dotnet build src/Core/Agents/Hazina.AgentFactory/Hazina.AgentFactory.csproj`
- Add as project reference if consumed from another project: `dotnet add <your>.csproj reference src/Core/Agents/Hazina.AgentFactory/Hazina.AgentFactory.csproj`

## Usage
- Add reference: `dotnet add <your>.csproj reference src/Core/Agents/Hazina.AgentFactory/Hazina.AgentFactory.csproj`
- Build: `dotnet build src/Core/Agents/Hazina.AgentFactory/Hazina.AgentFactory.csproj`

## API Reference
- XML docs generated on build: `bin/Debug/net8.0/Hazina.AgentFactory.xml`.
- Use IDE tooling or `dotnet doc`/Sandcastle/DocFX to render API docs if desired.
