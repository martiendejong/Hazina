# Hazina.Tools.Models

## Purpose
Shared domain models for Hazina tools.

## Getting Started
- Restore/build: `dotnet build src/Tools/Foundation/Hazina.Tools.Models/Hazina.Tools.Models.csproj`
- Add as project reference if consumed from another project: `dotnet add <your>.csproj reference src/Tools/Foundation/Hazina.Tools.Models/Hazina.Tools.Models.csproj`

## Usage
- Add reference: `dotnet add <your>.csproj reference src/Tools/Foundation/Hazina.Tools.Models/Hazina.Tools.Models.csproj`
- Build: `dotnet build src/Tools/Foundation/Hazina.Tools.Models/Hazina.Tools.Models.csproj`

## API Reference

- Use IDE tooling or `dotnet doc`/Sandcastle/DocFX to render API docs if desired.
