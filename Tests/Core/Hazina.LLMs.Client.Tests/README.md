# Hazina.LLMs.Client.Tests

## Purpose
Test suite for Hazina.LLMs.Client.

## Getting Started
- Run tests: `dotnet test Tests/Core/Hazina.LLMs.Client.Tests/Hazina.LLMs.Client.Tests.csproj`
- Build only (optional): `dotnet build Tests/Core/Hazina.LLMs.Client.Tests/Hazina.LLMs.Client.Tests.csproj`

## Usage
- Run entire suite: `dotnet test Tests/Core/Hazina.LLMs.Client.Tests/Hazina.LLMs.Client.Tests.csproj`
- Filter tests (example): `dotnet test Tests/Core/Hazina.LLMs.Client.Tests/Hazina.LLMs.Client.Tests.csproj --filter Category=Unit`
