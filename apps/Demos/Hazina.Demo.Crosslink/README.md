# Hazina.Demo.Crosslink

## Purpose
Demo app showing crosslinking/relationship generation.

## Getting Started
- Restore/build: `dotnet build apps/Demos/Hazina.Demo.Crosslink/Hazina.Demo.Crosslink.csproj`
- Add as project reference if consumed from another project: `dotnet add <your>.csproj reference apps/Demos/Hazina.Demo.Crosslink/Hazina.Demo.Crosslink.csproj`

## Usage
- Add reference: `dotnet add <your>.csproj reference apps/Demos/Hazina.Demo.Crosslink/Hazina.Demo.Crosslink.csproj`
- Build: `dotnet build apps/Demos/Hazina.Demo.Crosslink/Hazina.Demo.Crosslink.csproj`

## API Reference
- XML docs generated on build: `bin/Debug/net8.0/Hazina.Demo.Crosslink.xml`.
- Use IDE tooling or `dotnet doc`/Sandcastle/DocFX to render API docs if desired.
