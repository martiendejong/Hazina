# Hazina.Demo.Llama

## Purpose
Demo wiring Hazina against a LLaMA provider.

## Getting Started
- Restore/build: `dotnet build apps/Demos/Hazina.Demo.Llama/Hazina.Demo.Llama.csproj`
- Add as project reference if consumed from another project: `dotnet add <your>.csproj reference apps/Demos/Hazina.Demo.Llama/Hazina.Demo.Llama.csproj`

## Usage
- Add reference: `dotnet add <your>.csproj reference apps/Demos/Hazina.Demo.Llama/Hazina.Demo.Llama.csproj`
- Build: `dotnet build apps/Demos/Hazina.Demo.Llama/Hazina.Demo.Llama.csproj`

## API Reference
- XML docs generated on build: `bin/Debug/net8.0/Hazina.Demo.Llama.xml`.
- Use IDE tooling or `dotnet doc`/Sandcastle/DocFX to render API docs if desired.
