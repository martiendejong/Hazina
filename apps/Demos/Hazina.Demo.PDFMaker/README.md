# Hazina.Demo.PDFMaker

## Purpose
Demo creating PDFs with Hazina pipelines.

## Getting Started
- Restore/build: `dotnet build apps/Demos/Hazina.Demo.PDFMaker/Hazina.Demo.PDFMaker.csproj`
- Add as project reference if consumed from another project: `dotnet add <your>.csproj reference apps/Demos/Hazina.Demo.PDFMaker/Hazina.Demo.PDFMaker.csproj`

## Usage
- Add reference: `dotnet add <your>.csproj reference apps/Demos/Hazina.Demo.PDFMaker/Hazina.Demo.PDFMaker.csproj`
- Build: `dotnet build apps/Demos/Hazina.Demo.PDFMaker/Hazina.Demo.PDFMaker.csproj`
