# Hazina.Demo.Postgres

## Purpose
Demo of Postgres-backed Hazina flows.

## Getting Started
- Restore/build: `dotnet build apps/Demos/Hazina.Demo.Postgres/Hazina.Demo.Postgres.csproj`
- Add as project reference if consumed from another project: `dotnet add <your>.csproj reference apps/Demos/Hazina.Demo.Postgres/Hazina.Demo.Postgres.csproj`

## Usage
- Add reference: `dotnet add <your>.csproj reference apps/Demos/Hazina.Demo.Postgres/Hazina.Demo.Postgres.csproj`
- Build: `dotnet build apps/Demos/Hazina.Demo.Postgres/Hazina.Demo.Postgres.csproj`

## API Reference
- XML docs generated on build: `bin/Debug/net8.0/Hazina.Demo.Postgres.xml`.
- Use IDE tooling or `dotnet doc`/Sandcastle/DocFX to render API docs if desired.
