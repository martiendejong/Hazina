# Hazina.Demo.FolderToPostgres

## Purpose
Demo that ingests folder content into Postgres.

## Getting Started
- Restore/build: `dotnet build apps/Demos/Hazina.Demo.FolderToPostgres/Hazina.Demo.FolderToPostgres.csproj`
- Add as project reference if consumed from another project: `dotnet add <your>.csproj reference apps/Demos/Hazina.Demo.FolderToPostgres/Hazina.Demo.FolderToPostgres.csproj`

- Add reference: `dotnet add <your>.csproj reference apps/Demos/Hazina.Demo.FolderToPostgres/Hazina.Demo.FolderToPostgres.csproj`
