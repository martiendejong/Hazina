# Hazina.App.Windows

## Purpose
Windows desktop shell for Hazina.

## Getting Started
- Restore/build: `dotnet build apps/Desktop/Hazina.App.Windows/Hazina.App.Windows.csproj`
- Add as project reference if consumed from another project: `dotnet add <your>.csproj reference apps/Desktop/Hazina.App.Windows/Hazina.App.Windows.csproj`

## Usage
- Add reference: `dotnet add <your>.csproj reference apps/Desktop/Hazina.App.Windows/Hazina.App.Windows.csproj`
- Build: `dotnet build apps/Desktop/Hazina.App.Windows/Hazina.App.Windows.csproj`

## API Reference
- XML docs generated on build: `bin/Debug/net8.0/Hazina.App.Windows.xml`.
- Use IDE tooling or `dotnet doc`/Sandcastle/DocFX to render API docs if desired.
