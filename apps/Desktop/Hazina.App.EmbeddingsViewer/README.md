# Hazina.App.EmbeddingsViewer

## Purpose
Desktop viewer for embeddings produced by Hazina.

## Getting Started
- Restore/build: `dotnet build apps/Desktop/Hazina.App.EmbeddingsViewer/Hazina.App.EmbeddingsViewer.csproj`
- Add as project reference if consumed from another project: `dotnet add <your>.csproj reference apps/Desktop/Hazina.App.EmbeddingsViewer/Hazina.App.EmbeddingsViewer.csproj`

## Usage
